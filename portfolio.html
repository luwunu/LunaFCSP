Fish Tank Code


This coding project was based on malware, cyber security, and pass keys. We used a public key and a private key to decrypt certain bits of code to reveal a fish tank. The fish tank would show all levels of the water as "OK" which we had to fix so the code showed the true levels of the fish tank ao the fish wont die. This project was very confusing but it taught me lots.
